# nginx の覇者になりたい

## 環境

- macOS Catalina 10.15.4
- Docker 19.03.8

## コマンド一覧

### コンテナのビルド

```shell
$ make build
```

### サーバの起動

```shell
$ make start
```

### サーバの停止

```shell
$ make stop
```

### ページを開く

```shell
$ make open
```
